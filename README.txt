EL KARAOKE DE DIEGO - VERSION PREMIUM

Para subir:
1. Abrí el repositorio en GitHub.
2. Add file > Upload files.
3. Subí index.html y styles.css.
4. Si también aparece hero-video.mp4 y pesa poco, podés subirlo.
5. Commit changes.
6. Esperá 1 o 2 minutos y refrescá con Ctrl + F5.
