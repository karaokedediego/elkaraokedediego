WEB V2 - El Karaoke de Diego

Para actualizar GitHub:
1. Abrir el repositorio en GitHub.
2. Entrar a Code.
3. Click en Add file > Upload files.
4. Subir:
   - index.html
   - styles.css
   - carpeta assets completa
5. Commit changes.
6. Esperar 1 o 2 minutos y refrescar la página.

IMPORTANTE:
Si GitHub pregunta reemplazar archivos existentes, aceptar.
